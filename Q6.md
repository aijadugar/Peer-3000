## Method 1 - Software

The goal is to detect and remove glare regions before running detection.

### Steps

* Convert image to HSV color space
* Detect high-brightness, low-saturation pixels (white glare)
* Create a glare mask
* Apply inpainting to reconstruct the affected region

### Code Logic

```python id="k3h2l9"
hsv = cv2.cvtColor(frame, cv2.COLOR_BGR2HSV)

# Detect bright white glare regions
lower_glare = np.array([0, 0, 240])
upper_glare = np.array([180, 30, 255])

mask = cv2.inRange(hsv, lower_glare, upper_glare)

# Expand mask slightly
kernel = np.ones((5,5), np.uint8)
mask = cv2.dilate(mask, kernel)

# Inpaint glare regions
frame_clean = cv2.inpaint(frame, mask, 3, cv2.INPAINT_NS)
```
