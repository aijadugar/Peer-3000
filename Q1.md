## Question 1 - Visual Heuristics for “Readiness to Move”

### Approach

The system uses a rule-based deterministic pipeline where four independent checks are evaluated in parallel:

* Positioning
* Isolation
* Accessibility
* Stability

A pallet is considered **READY** only if all four checks pass.

```
READY = positioning AND isolation AND accessibility AND stability  
NOT_READY = any one check fails
```

---

## 1. Positioning

### Case 1 - With Blue Tape (Image 1 & 2)

The pallet must be properly aligned within the marked staging area.

Conditions:

* The center of the pallet should match the center of the tape within ±8%
* At least 85% of the pallet area should lie inside the tape region

**Code Logic**

```python
center_offset = abs(pallet_center - tape_center)
inside_area_ratio = pallet_area_inside_tape / total_pallet_area

positioning_pass = (center_offset < 0.08) and (inside_area_ratio > 0.85)
```

---

### Case 2 - Without Tape (Image 3)

Since no staging reference exists, alignment is inferred from pallet geometry.

Conditions:

* Pallet must be straight (not rotated)
* Fork pockets should be horizontal within ±5 degrees

**Code Logic**

```python
angle = detect_pallet_angle(frame)
fork_angle = detect_fork_orientation(frame)

positioning_pass = (abs(angle) < 5) and (abs(fork_angle) < 5)
```

---

## 2. Isolation

The pallet must have enough free space around it for safe robot movement.

Conditions:

* No nearby objects around the pallet
* No obstacle in front region (20% area in front of pallet)

**Code Logic**

```python
nearby_objects = detect_objects_around(pallet_bbox)
front_roi_clear = check_front_region_clear(frame, pallet_bbox, roi=0.2)

isolation_pass = (nearby_objects == 0) and front_roi_clear
```

---

## 3. Accessibility

The pallet should be physically accessible and safe to pick.

Conditions:

* Pallet should not be tilted (±4 degrees)
* Load height should be within acceptable limit (ratio < 3)

**Code Logic**

```python
tilt_angle = estimate_tilt(pallet_bbox)
height_ratio = load_height / pallet_height

accessibility_pass = (abs(tilt_angle) < 4) and (height_ratio < 3)
```

---

## 4. Stability

The load on the pallet should be balanced and not at risk of falling.

Conditions:

* Load center should align with pallet center
* Horizontal offset should be less than 12%

**Code Logic**

```python
offset = abs(load_center_x - pallet_center_x) / pallet_width

stability_pass = (offset < 0.12)
```
