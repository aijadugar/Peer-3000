## Question 4 - VLM Selection, Prompt and Outputs

### Approach

A lightweight Vision-Language Model is used only for resolving ambiguous cases where classical CV is uncertain.

---

## Selected Model

Chosen model: Moondream2

### Reason for Selection

* Runs in ~1.8 GB VRAM
* Fast inference (~400–600 ms)
* Works on Jetson-class hardware
* Supports structured JSON-style responses
* Good performance on spatial reasoning tasks

---

## Prompt Design

The prompt is designed to:

* Force strict JSON output
* Avoid free-form text
* Capture all required decision signals

### Prompt Used

```text id="q4prompt1"
You are an industrial robot perception system. Analyze the image of a warehouse pallet.

Answer the following questions in this exact JSON format and nothing else:

{
  "pallet_detected": true or false,
  "pallet_position": "centered" | "offset_left" | "offset_right" | "unknown",
  "staging_zone": "inside_tape" | "no_tape_visible" | "partially_inside",
  "clearance_around_pallet": "clear" | "obstructed" | "uncertain",
  "load_appears_stable": true or false,
  "glare_present": true or false,
  "ready_to_move": true or false,
  "reason": "<brief one-sentence explanation>"
}

Only output the JSON. Do not add any explanation before or after.
```

---

## Model Execution

```python id="q4code1"
def run_vlm(image_crop):
    prompt = PROMPT_TEXT
    response = moondream.generate(image_crop, prompt)

    return parse_json(response)
```
